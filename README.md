# Frontend of a Restaurant Menu Page

## Project Overview
This project is a frontend-only restaurant menu website developed using HTML, CSS, and JavaScript. It allows users to browse food items category-wise using a clean, responsive, and user-friendly interface.

## Objective
The objective of this project is to design and implement a restaurant menu webpage using core web technologies, focusing on layout design, interactivity, and usability without using any backend services.

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
- CSS Grid and Flexbox

## Features
- Hero section with welcome message and call-to-action
- Category-wise menu filtering using dropdown
- Card-based layout for menu items
- Responsive design for different screen sizes
- Footer with contact details and social links
